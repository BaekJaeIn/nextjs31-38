# nextjs 실습 파일

## Udemy에서 공부한 react, nextjs 실습 파일을 보관하는 repository입니다.

---

### 로컬 서버 실행 방법

`npm install`

`npm run dev`

34. **[보너스] 리액트 기초 및 실습 컴포넌트** (35-bonus-basics)
